# @dxos/readme

A utility for templating readme files

## Installation

```bash
pnpm i @dxos/readme
```

## Usage

The template is provided in `src/template` which can be executed with the `plate` executable from `@dxos/plate`. See the `dev` script in this package.
Execute with:

```bash
pnpm nx dev readme
```
