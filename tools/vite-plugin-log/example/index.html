DXOS Vite Log Plugin Example

This is a minimal test app for the @dxos/vite-plugin-log plugin.

Test Logging

Click the buttons below to trigger different log levels. Check the terminal and app.log file for output.

Instructions

  1. Run pnpm dev in this directory
  2. Open the browser console to see client-side logs
  3. Check the terminal for server-side log output
  4. Check app.log file for persisted logs
