# @dxos/halo-e2e

Public-API and integration tests for the `@dxos/halo` services, exercised through
`@dxos/halo-adapter-client` (the `@dxos/client`-backed implementation). Mirrors the coverage of
`@dxos/client-e2e` (identity, spaces, invitations, membership) against the new service API.
