`dx help`
=========

Display help for dx.

* [`dx help [COMMANDS]`](#dx-help-commands)

## `dx help [COMMANDS]`

Display help for dx.

```
USAGE
  $ dx help [COMMANDS] [-n]

ARGUMENTS
  COMMANDS  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for dx.
```
