# @dxos/graph

Basic graph API.

## Prior art

- [Graphology](https://graphology.github.io) (TS, tree-shakable, multiple packages for extensions)
- [Graphlib](https://github.com/dagrejs/graphlib) (mature, extensive)
- [tiny-graph](https://github.com/avoidwork/tiny-graph)
- levelgraph (LevelDB)
- oxigraph (Rust WASM)
- Neo4J
