# plugin-presenter

Markdown-to-slideshow presentation mode.

## Status

Stable.

## Description

Transforms a markdown document into a slide-by-slide presentation. Slides are delimited by horizontal rules and rendered full-screen for presentations.

## Features

- **Slide rendering**: Splits markdown on `---` into slides.
- **Full-screen mode**: Presents slides in distraction-free full-screen.
- **Keyboard navigation**: Arrow keys to advance/retreat slides.
- **Slide counter**: Displays current slide number and total.
- **Markdown support**: Full markdown formatting within slides.

## Schema

None (uses `org.dxos.type.document` from plugin-markdown).
