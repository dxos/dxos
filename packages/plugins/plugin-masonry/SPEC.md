# plugin-masonry

Responsive masonry grid layout for query results.

## Status

Stable.

## Description

Displays collections of objects in an adaptive masonry grid. Automatically reflows cards into columns based on available screen space.

## Features

- **Masonry layout**: Multi-column adaptive card grid.
- **Query binding**: Renders results from an ECHO query.
- **Card previews**: Shows per-type preview cards for each object.
- **Responsive reflow**: Adjusts column count to viewport width.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.masonry` — Masonry view with query and display config.
