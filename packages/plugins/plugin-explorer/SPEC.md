# plugin-explorer

Interactive hypergraph visualization for workspace relationships.

## Status

Labs (experimental).

## Description

Renders workspace objects and their relationships as an interactive force-directed graph, enabling users to visually explore connections across spaces.

## Features

- **Force-directed graph**: Dynamic layout of objects and links.
- **Pan and zoom**: Navigate large graphs fluidly.
- **Object filtering**: Show/hide node types.
- **Click to open**: Navigate to any object from the graph.
- **Translations**: Localizable UI strings.
