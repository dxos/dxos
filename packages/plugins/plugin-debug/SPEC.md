# plugin-debug

Developer toolkit for debugging and inspecting the workspace.

## Status

Stable (dev only).

## Description

Provides tools for troubleshooting plugins, generating test data, inspecting ECHO objects, and monitoring runtime events.

## Features

- **Object inspector**: Browse and inspect ECHO objects in raw form.
- **Event monitor**: Real-time stream of application events.
- **Test data generator**: Seed the workspace with sample objects.
- **Plugin diagnostics**: Inspect plugin state and capability graph.
- **Automation testing**: Trigger automation scenarios manually.
