# plugin-native

Native desktop platform integration for DXOS Composer.

## Status

Stable.

## Description

Provides access to native OS capabilities when running as a desktop application (Electron). Includes file dialogs, system notifications, and OS-level integrations.

## Features

- **Native file dialogs**: Open/save dialogs using OS file picker.
- **System notifications**: Push desktop notifications.
- **OS integrations**: Tray icon, window management, and app menu.
- **Protocol handlers**: Handle custom URL schemes.

## Schema

None (platform bridge plugin).
