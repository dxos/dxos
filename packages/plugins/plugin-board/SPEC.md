# plugin-board

Freeform canvas board for spatial content arrangement.

## Status

Labs (experimental).

## Description

Provides an infinite canvas where workspace objects can be arranged spatially. Supports freeform layout of cards, notes, and other objects with drag-and-drop positioning.

## Features

- **Infinite canvas**: Pan and zoom freeform workspace.
- **Object placement**: Position any workspace object on the board.
- **Card rendering**: Displays objects as cards with previews.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.board` — Board object containing spatially arranged items.
