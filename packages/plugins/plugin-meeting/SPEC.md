# plugin-meeting

Meeting management with notes, transcription, and AI summaries.

## Status

Labs (experimental).

## Description

Captures meeting notes with real-time transcription and generates AI-powered summaries with action items. Meetings are stored in ECHO and linked to calendar events.

## Features

- **Meeting notes**: Rich text editor for capturing discussion.
- **Live transcription**: Real-time voice-to-text via plugin-transcription.
- **AI summaries**: Automatic post-meeting summary with action items.
- **Timestamps**: Synchronized note timestamps with audio.
- **Calendar integration**: Link meetings to calendar events.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.meeting` — Meeting object with notes, transcript, and metadata.
