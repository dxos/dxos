# plugin-markdown

Full-featured collaborative markdown editor.

## Status

Stable.

## Description

Rich text markdown editor with real-time collaboration, inline comments, and AI-assisted editing. Documents are stored in ECHO as CRDT text objects.

## Features

- **Rich text editing**: Bold, italic, headings, lists, code blocks, and more.
- **Real-time collaboration**: Multiple users edit simultaneously via ECHO CRDT.
- **Inline comments**: Annotate specific text ranges with threaded comments.
- **AI editing**: AI-powered suggestions, rewrites, and completions.
- **Keyboard shortcuts**: Full markdown editing key bindings.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.document` — Markdown document with CRDT text content.
