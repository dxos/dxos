# plugin-observability

Application telemetry and performance monitoring.

## Status

Stable.

## Description

Collects and reports application metrics, logs, and traces for monitoring system health, diagnosing issues, and analyzing usage patterns.

## Features

- **Metrics collection**: Track performance counters and resource usage.
- **Error reporting**: Capture and report runtime errors.
- **Usage analytics**: Anonymized feature usage tracking.
- **Trace correlation**: Link events across async operations.
- **Opt-in/opt-out**: User controls for telemetry consent.

## Schema

None (telemetry data sent to external service).
