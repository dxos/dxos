# plugin-client

Core DXOS client connectivity and identity management.

## Status

Stable.

## Description

Provides the foundational DXOS client, managing peer-to-peer networking, identity, authentication, and real-time ECHO synchronization across devices.

## Features

- **Identity management**: Create and manage DXOS user identities.
- **P2P networking**: Establishes connections to peers and spaces.
- **ECHO sync**: Real-time data synchronization across devices and users.
- **Authentication**: Handles device authorization and space invitations.
- **Client lifecycle**: Initializes and tears down the DXOS client.

None (infrastructure plugin).
