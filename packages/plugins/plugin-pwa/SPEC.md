# plugin-pwa

Progressive Web App support for DXOS Composer.

## Status

Stable.

## Description

Enables Composer to function as a Progressive Web App with offline capabilities, home screen installation, and app-like experience in the browser.

## Features

- **Service worker**: Caches assets for offline use.
- **Install prompt**: Prompts users to install to home screen.
- **Offline mode**: Core workspace functions available without internet.
- **Update notifications**: Alerts user when a new version is available.

None (platform infrastructure plugin).
