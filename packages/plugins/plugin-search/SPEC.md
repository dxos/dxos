# plugin-search

Full-text search across all workspace content.

## Status

Labs (experimental).

## Description

Indexes and searches all objects across spaces and types. Returns ranked results with previews and navigates directly to matching objects.

## Features

- **Full-text index**: Indexes all text content in ECHO objects.
- **Instant results**: Low-latency search-as-you-type.
- **Cross-space search**: Searches across all accessible spaces.
- **Result ranking**: Relevance-ranked results with previews.
- **Type filtering**: Filter results by object type.
- **Keyboard navigation**: Arrow keys to navigate results.

## Schema

None (indexes existing ECHO objects).
