# plugin-theme

Core theming engine for visual styling.

## Status

Stable.

## Description

Provides the foundational theme system for Composer, including light/dark mode switching, color palette management, and TailwindCSS configuration.

## Features

- **Light/dark mode**: Toggle between light and dark themes.
- **Color system**: Consistent semantic color tokens across the UI.
- **TailwindCSS**: Theme values exposed as Tailwind utility classes.
- **Plugin theming**: Other plugins inherit and extend the base theme.

## Schema

None (theme configuration only).
