# plugin-chess

Collaborative chess game for DXOS Composer.

## Status

Labs (experimental).

## Description

Embeds a fully playable chess game into the workspace. Game state is stored in ECHO and synchronized in real-time between players.

## Features

- **Real-time play**: Moves sync instantly via ECHO to all participants.
- **Game state**: Full chess rules including castling, en passant, and promotion.
- **Board UI**: Interactive chessboard with piece drag-and-drop.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.chess` — Chess game state object.
