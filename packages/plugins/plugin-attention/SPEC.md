# plugin-attention

Attention and focus management for DXOS Composer.

## Status

Stable.

## Description

Tracks which objects and spaces have user attention, enabling focus-aware UI behaviors and notifications.

## Features

- **Attention tracking**: Monitors which object is currently focused.
- **Focus events**: Emits events when attention shifts between objects.
- **Integration**: Used by other plugins to drive context-sensitive behavior.
