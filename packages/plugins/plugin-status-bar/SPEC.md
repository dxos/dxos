# plugin-status-bar

Persistent bottom status bar with workspace state and quick actions.

## Status

Stable.

## Description

Renders a persistent bottom bar showing connection status, sync state, notifications, and quick-access action buttons.

## Features

- **Connection status**: Displays P2P network and ECHO sync state.
- **Notifications**: Shows workspace alerts and system messages.
- **Quick actions**: Common commands accessible without navigation.
- **Plugin contributions**: Other plugins can contribute status bar items.
