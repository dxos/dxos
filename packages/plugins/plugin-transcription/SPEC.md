# plugin-transcription

Real-time voice-to-text transcription service.

## Status

Labs (experimental).

## Description

Converts audio input to text in real-time using speech recognition. Integrates with plugin-meeting and plugin-thread to capture spoken content into workspace objects.

## Features

- **Live transcription**: Real-time speech-to-text from microphone input.
- **Speaker detection**: Identifies and labels speakers when possible.
- **Transcript storage**: Saves transcripts as searchable ECHO text.
- **Meeting integration**: Attaches transcripts to meeting objects.
- **Language support**: Multiple language recognition models.

## Schema

None (transcripts stored as text in parent objects).
