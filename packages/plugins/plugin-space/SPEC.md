# plugin-space

Core workspace space management.

## Status

Stable.

## Description

Manages the lifecycle of DXOS spaces — the top-level collaborative containers. Handles creation, sharing, invitations, and access control for all workspace content.

## Features

- **Space creation**: Create new spaces with custom names and icons.
- **Invitations**: Invite collaborators via links or QR codes.
- **Access control**: Manage member roles and permissions.
- **Object creation**: Unified dialog for creating typed objects in a space.
- **Space settings**: Configure space metadata and membership.
- **Migration**: Tools for migrating space data between versions.

## Schema

None (spaces are native DXOS constructs).
