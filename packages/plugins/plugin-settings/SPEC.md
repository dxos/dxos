# plugin-settings

Workspace settings management.

## Status

Stable.

## Description

Provides a centralized settings panel where users and plugins can register and manage configuration options.

## Features

- **Settings panel**: Unified UI for all workspace settings.
- **Plugin settings**: Each plugin can register its own settings section.
- **Persistence**: Settings stored and synced via ECHO.

## Schema

None (settings stored in plugin state).
