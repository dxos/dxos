# plugin-files

Local filesystem bridge for DXOS Composer.

## Status

Labs (experimental).

## Description

Allows users to open local files directly in the workspace and edit them in place. Changes sync bidirectionally between the local filesystem and ECHO.

## Features

- **File open**: Open local files via native file picker.
- **Bidirectional sync**: Edits in Composer reflect on disk and vice versa.
- **File watching**: Detects external changes to open files.
- **Multi-format**: Supports text, markdown, and other file types.

## Schema

None (files referenced by path).
