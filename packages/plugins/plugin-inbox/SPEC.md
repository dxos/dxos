# plugin-inbox

Unified inbox for email, calendar, and contacts.

## Status

Labs (experimental).

## Description

Aggregates email, calendar events, and contact information into a unified workspace view. Connects to external mail and calendar services for bidirectional sync.

## Features

- **Email view**: Read and manage messages from connected mailboxes.
- **Calendar view**: Browse and create calendar events.
- **Contact management**: View and organize contacts.
- **External sync**: Connect Gmail, CalDAV, and other services.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.mailbox` — Mailbox configuration and message store.
- `org.dxos.type.calendar` — Calendar object with event collection.
