# plugin-excalidraw

Professional diagramming with hand-drawn style via Excalidraw.

## Status

Labs (experimental).

## Description

Embeds the Excalidraw diagramming library for creating flowcharts, wireframes, and technical diagrams with a hand-drawn aesthetic. Diagrams are stored in ECHO and sync in real-time.

## Features

- **Rich shape library**: Rectangles, arrows, text, and freehand drawing.
- **Hand-drawn style**: Sketchy rendering for informal diagrams.
- **Real-time collaboration**: ECHO-backed sync across users.
- **Export**: Download diagrams as PNG or SVG.
- **Translations**: Localizable UI strings.

## Schema

None (diagram data embedded in ECHO object).
