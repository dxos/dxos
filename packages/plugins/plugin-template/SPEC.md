# plugin-template

Reusable object templates for workspace content.

## Status

Stable.

## Description

Allows users to define and instantiate templates for documents, tables, and other objects. Templates pre-configure content and structure for common use cases.

## Features

- **Template creation**: Save any object as a reusable template.
- **Template library**: Browse and select templates when creating objects.
- **Pre-configured content**: Templates include starter content and settings.
- **Type-specific templates**: Templates scoped to specific object types.

## Schema

- `org.dxos.type.data` — Generic template data container.
