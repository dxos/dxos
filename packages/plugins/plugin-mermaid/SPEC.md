# plugin-mermaid

Diagram generation from Mermaid text syntax.

## Status

Stable.

## Description

Renders diagrams defined using Mermaid's text-based syntax. Diagrams update live as the source text changes, and can be embedded in markdown documents.

## Features

- **Live preview**: Renders Mermaid diagrams as SVG in real-time.
- **Multiple diagram types**: Flowchart, sequence, Gantt, class, ER, and more.
- **Markdown embedding**: Embed diagrams in fenced code blocks.
- **Export**: Download rendered diagrams as SVG or PNG.

## Schema

None (diagram source stored as text in ECHO).
