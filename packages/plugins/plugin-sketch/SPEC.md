# plugin-sketch

Lightweight digital whiteboard for freehand drawing.

## Status

Stable.

## Description

A simple whiteboard canvas for freehand sketches, shapes, and annotations. Backed by tldraw and stored in ECHO for real-time collaborative drawing.

## Features

- **Freehand drawing**: Pen and brush tools for freehand input.
- **Shapes**: Rectangles, ellipses, arrows, and text.
- **Real-time collaboration**: Shared drawing canvas via ECHO.
- **Selection and editing**: Move, resize, and style objects.
- **Export**: Download canvas as image.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.sketch` — Sketch canvas with drawing content.
- `org.dxos.type.canvas` — Generic canvas object.
