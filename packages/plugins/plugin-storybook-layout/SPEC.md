# plugin-storybook-layout

Storybook development layout for component testing.

## Status

Dev/internal.

## Description

Provides a specialized layout for Storybook stories, enabling component development and design system exploration within the Composer plugin framework.

## Features

- **Storybook surface**: Renders plugin surfaces in isolated Storybook context.
- **Theme integration**: Applies workspace theme to Storybook canvas.
- **Layout wrapper**: Wraps stories in Composer layout chrome.
