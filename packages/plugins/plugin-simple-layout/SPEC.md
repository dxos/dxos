# plugin-simple-layout

Minimal layout for simplified UI contexts.

## Status

Stable.

## Description

A stripped-down layout plugin for rendering content in popover windows, embedded views, and other constrained UI contexts without full Composer chrome.

## Features

- **Minimal chrome**: No sidebar, navtree, or status bar.
- **Popover support**: Renders content in popover/dialog contexts.
- **Content focus**: Single-object view optimized for focus.

None (layout infrastructure plugin).
