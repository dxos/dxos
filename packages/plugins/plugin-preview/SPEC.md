# plugin-preview

Rich preview panel for workspace objects.

## Status

Stable.

## Description

Renders inline previews of workspace objects without opening them in a full editor. Used in sidebars, hover cards, and summary views.

## Features

- **Inline preview**: Renders object content in compact form.
- **Multi-type support**: Previews for documents, tables, images, and more.
- **Form card**: Structured property display for typed objects.
- **Read-only**: Preview-only rendering with no edit capability.

## Schema

None (renders any ECHO object).
