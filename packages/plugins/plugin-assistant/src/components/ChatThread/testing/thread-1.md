### Markdown

Markdown is a lightweight markup language used to format plain text in a simple and readable way.

It allows you to create structured documents using conventions for headings, lists, emphasis (bold/italic), links, images, code, blockquotes, tables, and horizontal rules.

Markdown documents look like this

```
# Markdown

This is a plain text.
```
