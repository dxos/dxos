# plugin-sheet

Full-featured spreadsheet with formulas and AI integration.

## Status

Stable.

## Description

A spreadsheet application with over 400 built-in formulas, custom JavaScript function support, and AI agent integration. Sheet data is stored in ECHO for real-time collaboration.

## Features

- **Cell editing**: Formula bar with expression evaluation.
- **400+ formulas**: Full formula library (math, text, date, logic, lookup).
- **Custom functions**: Register JavaScript functions via plugin-script.
- **Named ranges**: Define and reference named cell ranges.
- **Formatting**: Cell formatting, borders, and alignment.
- **AI integration**: AI can read/write cells and run analyses.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.sheet` — Spreadsheet object with cell data and formula definitions.
