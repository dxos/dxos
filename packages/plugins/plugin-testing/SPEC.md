# plugin-testing

Testing utilities and layout for plugin development.

## Status

Dev/internal.

## Description

Provides testing infrastructure, mock capabilities, and a Storybook-compatible layout for developing and testing plugins in isolation.

## Features

- **Test layout**: Renders plugin surfaces in a test harness.
- **Mock capabilities**: Provides stub implementations of plugin dependencies.
- **Storybook integration**: Works alongside plugin-storybook-layout.
