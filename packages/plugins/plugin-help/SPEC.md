# plugin-help

Built-in help system with documentation and keyboard shortcuts.

## Status

Stable.

## Description

Provides contextual help, keyboard shortcut references, feature guides, and links to support resources within the workspace.

## Features

- **Keyboard shortcuts**: Browse all available key bindings.
- **Feature guides**: Inline documentation for workspace features.
- **Command palette**: Search and execute commands.
- **Support links**: Direct access to docs and community resources.
