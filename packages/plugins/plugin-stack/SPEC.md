# plugin-stack

Curated collections of workspace objects.

## Status

Stable.

## Description

Organizes groups of workspace objects into ordered collections (stacks). Collections can contain sections, mix object types, and be displayed in various layout views.

## Features

- **Collections**: Group any workspace objects into an ordered list.
- **Sections**: Divide collections into named sections.
- **Mixed types**: Combine documents, tables, sketches, etc. in one collection.
- **Layout views**: Display collections in list, grid, or masonry layouts.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.stack-view` — Collection/stack view with ordered sections.
