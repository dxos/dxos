# plugin-thread

Chat, threaded comments, and video conferencing.

## Status

Stable.

## Description

Multi-modal communication plugin supporting direct messaging, threaded comments on workspace objects, and video/audio conferencing.

## Features

- **Chat channels**: Real-time text messaging in named channels.
- **Threaded comments**: Attach comment threads to any workspace object.
- **Video conferencing**: WebRTC-based video/audio calls.
- **Reactions**: Emoji reactions on messages.
- **Message history**: Persistent message log via ECHO.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.channel` — Chat channel with message history.
