# @dxos/plugin-sequencer

Music sequencer plugin for DXOS Composer. Defines a `Score` containing `Track`
metadata and per-track `Sequence` objects whose `Note`s are edited on a 2D
piano-roll style grid (pitch on the y-axis, time on the x-axis), built on
`@dxos/react-ui-canvas` `CellGrid`.

See [`PLUGIN.mdl`](./PLUGIN.mdl) for the full specification.
