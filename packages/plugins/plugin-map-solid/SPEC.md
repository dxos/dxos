# plugin-map-solid

SolidJS map surface for DXOS Composer.

## Status

Labs (experimental).

## Description

Provides a SolidJS-based rendering surface for the map plugin, enabling map views within SolidJS application contexts.

## Features

- **SolidJS surface**: Renders the map plugin UI using SolidJS reactivity.
- **Same data model**: Shares the `org.dxos.type.map` schema with plugin-map.

## Schema

None (uses `org.dxos.type.map` from plugin-map).
