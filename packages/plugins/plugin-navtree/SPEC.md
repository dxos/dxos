# plugin-navtree

Hierarchical sidebar navigation tree.

## Status

Stable.

## Description

Renders a collapsible tree-based navigation sidebar for browsing spaces, folders, and objects. Provides the primary navigation structure of Composer.

## Features

- **Tree navigation**: Expandable/collapsible hierarchy of spaces and objects.
- **Context menus**: Right-click actions on any node.
- **Drag-and-drop**: Reorder and reparent objects.
- **Keyboard navigation**: Arrow key traversal and shortcuts.
- **Commands dialog**: Global command palette integration.

## Schema

None (navigation state derived from app graph).
