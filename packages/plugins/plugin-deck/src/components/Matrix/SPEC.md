# Matrix

- `Matrix` is an experimental alternative to `DeckMain`.
- It is similar to `Mosaic.Stack`; in the plugin we specify the `Tile` to wrap `Plank` objects.
- It is a radix-style container that has a horizontally scrolling collection of `Plank` objects.
- The right-most `Plank` serves as the "companion" area for the `Plank` to its left.
