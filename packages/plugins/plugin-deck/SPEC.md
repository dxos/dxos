# plugin-deck

Flexible workspace layout system with tabs, splits, and panels.

## Status

Stable.

## Description

Manages the overall UI layout of Composer, enabling users to arrange views in tabs, side-by-side splits, and floating panels with drag-and-drop reordering.

## Features

- **Tab management**: Open multiple objects in tabbed views.
- **Split views**: Horizontal and vertical panel splits.
- **Drag-and-drop**: Rearrange panels and tabs freely.
- **Layout persistence**: Saves layout state per space.
- **Plank system**: Core "deck of planks" layout metaphor.

## Schema

None (layout state managed internally).
