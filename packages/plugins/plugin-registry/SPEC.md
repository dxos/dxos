# plugin-registry

Plugin management and discovery for DXOS Composer.

## Status

Stable.

## Description

Provides a UI for discovering, enabling, disabling, and configuring workspace plugins. Users can browse available plugins and customize their Composer installation.

## Features

- **Plugin browser**: List all available plugins with descriptions.
- **Enable/disable**: Toggle plugins on or off.
- **Plugin details**: View plugin metadata, screenshots, and tags.
- **Settings integration**: Links to per-plugin settings panels.

## Schema

None (plugin state stored in settings).
