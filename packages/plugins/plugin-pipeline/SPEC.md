# plugin-pipeline

Pipeline-based task and milestone tracking.

## Status

Labs (experimental).

## Description

Organizes work into structured pipelines with stages, tasks, and milestones. Tracks progress and coordinates team activities across multiple workstreams.

## Features

- **Pipeline stages**: Define sequential workflow stages.
- **Task tracking**: Assign and track tasks within each stage.
- **Milestone markers**: Flag key deliverables in the pipeline.
- **Progress visualization**: Visual pipeline progress overview.
- **Translations**: Localizable UI strings.

## Schema

None (pipeline data stored via ECHO tables).
