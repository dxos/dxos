# plugin-kanban

Visual kanban board for project and workflow management.

## Status

Stable.

## Description

Displays table data as a kanban board organized by column/status values. Cards can be dragged between columns and automations can be triggered on status changes.

## Features

- **Column grouping**: Groups records by a chosen field value.
- **Drag-and-drop**: Move cards between columns to update status.
- **Custom columns**: Define column values and ordering.
- **Automation triggers**: Fire actions when cards change columns.
- **Data binding**: Backed by an ECHO table/view.
- **Translations**: Localizable UI strings.

## Schema

- `org.dxos.type.kanban` — Kanban board configuration with column definitions.
