# ECHO Graph

The component below implements a dynamic graph view of an ECHO database.

## Try it out.

This can be generated using simple hooks to create the model.

```tsx
export const App = () => {
  const client = useClient();
  const [space, setspace] = useState<space>();
  const model = useGraphModel(space);

  return (
    <EchoGraph model={model} />
  );
};
```
