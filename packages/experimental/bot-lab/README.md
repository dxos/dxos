# @dxos/bot-lab

## Set-up

1. Start Docker (e.g., Docker Desktop).

2. Run the proxy:

```bash
cd bot-lab
pnpm run proxy
```

## Development

Build image of NodeJS server.

```bash
pnpm run build:image
````

NOTE: Delete running containers via the Docker Desktop or cli.


## Installation

```bash
pnpm i @dxos/bot-lab
```
