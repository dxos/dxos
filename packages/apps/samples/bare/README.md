# @dxos/bare

This app was created with the DXOS `bare` application template.

Run the app with `pnpm`:

```bash
pnpm install
pnpm serve
```

Build the app to the `out` folder:

```bash
pnpm build
```

Deploy the app to a [DXOS Kube](https://docs.dxos.org/guide/kube/quick-start):

```
pnpm deploy
```
