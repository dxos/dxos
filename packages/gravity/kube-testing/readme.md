# @dxos/kube-testing

KUBE Testing.

## Installation

```bash
pnpm i @dxos/kube-testing
```

## Usage

1. Specify path to kube repo in constant `PATH_TO_KUBE_REPO` at `src/run-test-signal.ts`
2. Run tests with `pnpm run run-tests`
