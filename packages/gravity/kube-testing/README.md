
# @dxos/kube-testing

KUBE Testing.

## Prerequisites

- Run redis

```bash
brew install redis
redis-server
```

## Simple Usage
```bash
pnpm run-tests <testname>
```

tests currently include:

* `signal`
* `transport`
* `echo`
* `replication`

For additional usage, `pnpm run-tests --help`
