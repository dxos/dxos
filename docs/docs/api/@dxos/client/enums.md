# Enumerations
### [`ItemFilterDeleted`]()

Controls how deleted items are filtered.

Values:
- `HIDE_DELETED` Do not return deleted items. Default behaviour.
- `SHOW_DELETED` Return deleted and regular items.
- `SHOW_DELETED_ONLY` Return only deleted items.
