```bash
$ yarn

```

## Troubleshooting

- Edit `node_modeuls/gatsby-theme-apollo-docs/node-config.js` to debug.
- There must be an `content/index.md[x]` file.
- There must be AT LEAST one `.md` AND one `.mdx` file.
