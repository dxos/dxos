To create or join a [space](/echo/typescript/api) in ECHO, the user must first establish a [HALO identity](/halo/introduction). This can be done by creating a **new identity** or accepting a **device join invitation**.

## Receiving a device join invitation

We often want to add a device to the same [HALO identity](/halo/introduction).
You can accept a device join invitation by providing the invitation code and the auth code to the DXOS client.

<Code code={deviceJoin.slice(34)} lang="ts" />

<Aside type="tip" title="Device join invitation">
  Your client instance needs to be [configured](/echo/typescript/config) correctly to talk to a [signaling service](/additional-resources/glossary#signaling-service).
</Aside>
